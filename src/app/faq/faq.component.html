Administrator How-To:


Adding a User

Step 1: Navigate to the Users List page - the link is located in the top right of the page under the profile icon when you are signed in as an Administrator.



Step 2: On the User List Page, click the "Add A New User" button. This will bring you to the User Creation Page.


Step 3: On the User Creation Page, fill out the form with the New User's Information. Note: Ensure all fields are filled out and in the proper format or you will receive an error. The submit button will not be active until all errors are resolved.



Step 4: Hit the Submit button at the bottom of the form. You will then be redirected back to the User List page where you should see your newly created User!

Disabling a User

Step 1: Navigate to the Users List page - the link is located in the top right of the page under the profile icon when you are signed in as an Administrator.



Step 2: Click the trash can icon next to the user you wish to disable.



Step 3: After clicking the trash can icon, a dialog box will ask you to confirm that you want to disable the user. Click OK.

Step 4: After clicking OK, you should see a User Disabled Successfully message and then the page will refresh after a few seconds and the user that was disabled should no longer be visible.
