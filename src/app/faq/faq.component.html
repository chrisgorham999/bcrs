Administrator How-To:


Adding a User

Step 1: Navigate to the Users List page - the link is located under the profile icon when you are signed in as an Administrator.



Step 2:On the User List Page, click the "Add A New User" button. This will bring you to the User Creation Page.

Step 3:On the User Creation Page, fill out the form with the New User's Information. Note: Ensure all fields are filled out and in the proper format or you will receive an error. The submit button will not be active until all fields are filled in and have zero errors.
